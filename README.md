# Port scanner 
  - Created using socket module
  - Version 1: Returns open ports
  - Version 2: Returns open ports along with services running on open ports
<br>

## Arguments
  - Positional:
    - IP address
  - Optional:
    - Port range
<br>

## Usage

```
python portscanner.py 127.0.0.1 -p 1-100
```
